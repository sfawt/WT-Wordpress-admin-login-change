== Description ==

*WT Admin Login*" adlı eklenti, WordPress web sitenizin yönetici giriş URL'sini istediğiniz herhangi bir URL'ye güvenli bir şekilde değiştirmenizi sağlayan kullanımı kolay bir WordPress eklentisidir. 

Eklentinin basit iki adımlık süreci var. WordPress yönetici giriş URL'nizin güvenliğini saniyeler içinde sağlar ve tüm bunlar herhangi bir kodlama yapmadan gerçekleşir.

**SKötü amaçlı saldırılara karşı güvenlik** - Herhangi biri WordPress web sitenizin varsayılan giriş sayfasını bulabilir ve kolaylıkla saldırılara açık hale gelecektir. "wt-admin Login" eklentisi, yönetici giriş URL'nizi değiştirmenize ve herhangi bir kullanıcıyı başka bir URL'ye yönlendirmenize olanak tanır.

"wt-admin Login" eklentisi, çekirdek dosyaları yeniden adlandırmaz veya değiştirmez, bunun yerine sadece sayfa isteklerini yakalar ve başka bir URL'ye yönlendirir.

= Neden WT-admin Login eklentisini kullanmalısınız? =
- WordPress web sitenizi saldırılara karşı koruyun..
- Kodlama gerektirmeyen kolay iki adımda URL değiştirin.
- WordPress web sitenizi hackerlara ve siber saldırılara karşı kolayca güvence altına alın.
- Yalnızca güvendiğiniz kişilere erişim sağlayın.
- Admin sayfanıza sizin haricinizdeki girişleri engelleyin. 
- Kodlama ile bir URL oluşturmaktan çok daha kolay ve hatasız.

== Özellikler ==

**Yönlendirme Özel Alanı**

wp-admin / wp-login.php / admin gibi URL'leri kullanan kullanıcıları istediğiniz farklı bir sayfaya yönlendirebilirsiniz.

= Nasıl Çalışır? =

Eklentiler sayfasından "Ayarlar" butonunu kullanarak yada Kalıcı bağlantılar alanından rahatlıkla düzenleme yapabilirsiniz.

1. Adım: Yeni Giriş URL belirle
2. Adım: Hatalı giriş için URL belirle

**Note** - Bu eklentiyi etkinleştirdikten sonra wp-admin dizini ve wp-login.php sayfası kullanılamaz hale gelir, bu nedenle URL'yi yer imi olarak kaydetmeli veya hatırlamalısınız. Bu eklentiyi devre dışı bırakmak, sitenizi tam olarak önceki durumuna getirir.

== Destek ==
** Problem mi yaşıyorsunuz? ** wt.net.tr adresinden bizimle iletişime geçin.

== Yükleme adımları ==

1. Eklentiler sayfasına gidin ve "Yeni Ekle" butonuna tıklayın.
2. Bilgisayarınıza indirdiğiniz "wt-admin-login.zip" dosyasını seçin.
3. Eklentiyi Yükleyin ve aktif edin.
4. Dilerseniz eklentiler sayfasından Ayarlar butonunu kullanın veya Ayarlar > Kalıcı Bağlantılar adımına gidin."
5. Yeni belirleyeceğiniz URL'ler için "Giriş URL'nizi değiştirin." alanına gelin ve bu alandan dilediğiniz URL'yi ekleyin.
